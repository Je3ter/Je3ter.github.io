F - Octopus

题意

xxx 坐标轴上有 nnn 个物品，位置为 x1,x2,⋯ ,xnx_1,x_2,\cdots,x_nx1​,x2​,⋯,xn​。现在有一只章鱼，处于坐标 kkk 处，有长度为 l1,l2,⋯ ,lnl_1,l_2,\cdots,l_nl1​,l2​,⋯,ln​ 的手臂，第 iii 条手臂可以抓取位置在 [k−li,k+li][k-l_i,k+l_i][k−li​,k+li​] 的物品物品，每条手臂只能用于抓取一个物品。问有多少个位置满足能够抓到所有的物品。

数据范围：1≤n≤200,−1018≤x1<x2<⋯<xn≤1018,1≤l1≤l2≤⋯≤ln≤10181\leq n\leq 200,-10^{18}\leq x_1<x_2<\cdots<x_n\leq 10^{18},1\leq l_1\leq l_2\leq\cdots\leq l_n\leq 10^{18}1≤n≤200,−1018≤x1​<x2​<⋯<xn​≤1018,1≤l1​≤l2​≤⋯≤ln​≤1018。

题解

感觉很educational的题目。

首先，考虑对于一个固定的位置，如何check：答案很容易，每次一定是抓取离它最近的物品，如果当前这条手臂够不到，那么一定不行，否则就可以。只需要将所有物品分为左侧和右侧，每次比较一下是左边最近的离得近还是右边最近的离得近就行了。时间复杂度为 O(n)O(n)O(n)。

但是，由于数据范围很大，不能对每个位置进行检验。可以这样考虑：由于物品的个数不太多，所以位置合法性的变化点不会很多，我们来考虑一下什么时候会变化。也就是下面两种情况：

  • k0k_0k0​ 合法，k0+1k_0+1k0​+1 不合法

    则必然有 k0−lj≤xi≤k0+ljk_0-l_j\leq x_i\leq k_0+l_jk0​−lj​≤xi​≤k0​+lj​，且 k0+1−lj>xik_0+1-l_j>x_ik0​+1−lj​>xi​ 或 k0+lj+1<xik_0+l_j+1<x_ik0​+lj​+1<xi​（后一种情况是不可能的）。两个联立，可以得到 k0=xi+ljk_0=x_i+l_jk0​=xi​+lj​。

  • k0k_0k0​ 不合法，k0+1k_0+1k0​+1 合法

    类似地，k0=xi−lj−1k_0=x_i-l_j-1k0​=xi​−lj​−1。

所以我们一共得到了 2n22n^22n2 个分界点，只需要对这些点进行check就可以了。若一个分界点合法，则从它左侧一直到前一个分界点的位置都是合法的。

G - Typical Path Problem

题意

给出一张 nnn 个点 mmm 条边的无向连通图。给出 a,b,ca,b,ca,b,c 三个点，询问是否有一条 aaa 到 ccc 的简单路径经过点 bbb。

数据范围：3≤n≤2×105,n−1≤m≤min⁡(n×(n−1)2,2×105)3\leq n\leq 2\times10^5,n-1\leq m\leq \min(\dfrac{n\times(n-1)}{2},2\times 10^5)3≤n≤2×105,n−1≤m≤min(2n×(n−1)​,2×105)。

题解

圆方树板题。建出圆方树以后判断一下 aaa 到 ccc 的树上路径经过的方点相邻的圆点是否包含 bbb 即可。
