AtCoder Regular Contest 162

C - Mex Game on Tree

题意

给出一棵树，其中一些点标有数字。Alice和Bob轮流给剩下的点标数字，如果最终有一棵子树的所有结点 mex\mathrm{mex}mex 为 kkk，则Alice胜，否则Bob胜。要求判断两人最优操作下谁能获胜。

题解

很直观的想法是Bob每次操作都会填入 kkk，而若一棵子树中存在一个结点数字为 kkk，那么这棵子树的 mex\mathrm{mex}mex 不再可能为 kkk。那么，如果Alice不能在一步以内使得某棵子树的 mex\mathrm{mex}mex 为 kkk 且不再有结点没有填数字，Bob总可以破坏掉这棵子树。

答案就很显然了。我们寻找是否存在 mex\mathrm{mex}mex 为 k−1k-1k−1 且有 111 个结点未填数或 mex\mathrm{mex}mex 为 kkk 且有 000 或 111 个结点未填数的子树，若存在则Alice获胜，否则Bob获胜。

这可以利用树上启发式合并在 O(nlog⁡n)O(n\log n)O(nlogn) 的时间复杂度内完成。
